## Line Graph for Time Trend

### Non-time-series Graph
Preparation Code
```
# Sample Data
library(wooldridge)
B <- barium
```
**Actual Code**
```
plot(B$t, B$gas,
     main = "Line Graph of Gasoline Production Over Time",
     ylab = "Gasoline Production", xlab = "Time Trend")
lines(B$t, B$gas)
```
